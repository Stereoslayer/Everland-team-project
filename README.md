# Командный проект студентов факультета WEb+ 15 когорты -  Главная страница для нового сайта [Everland](https://evland.ru/about-project/) 
 

### **Опроекте:**
[Макет](https://www.figma.com/file/59a1PXM1KLWN0hWWMl1Kni/Everland-(Copy)?node-id=300%3A539) был подготовлен выпускницей факультета дизайна Практикума. Дизайн разработан с учетом пользователей с нарушением зрения и моторики.
Проект адаптирован под разные экраны от 320px и больше с помощью Grid Layout и Flexbox.
Спроектирвоан с помощью БЭМ.

Файловая структура: Nested.

Используемые технологии: HTML, CSS, JavaScript.
